# 진심모의수능 예약 웹사이트

📝 '진심 모의수능'을 온라인으로 예약하고, 좌석을 확인하며, 관리자 페이지에서 예약 데이터를 관리할 수 있는 웹 애플리케이션입니다.

---

## 📌 주요 기능

- 사용자 회원가입 및 로그인
- 모의수능 시험 목록 조회 및 예약
- 좌석 현황 확인
- 결제 기능 (예정)
- 문자 발송 (예정)
- 예약 취소 및 환불
- 관리자 페이지를 통한 예약 데이터 관리

---

## 🚀 기술 스택

- **Frontend**: React, Context API, React Router
- **Backend**: (예정) Node.js, Express
- **Database**: (예정) MongoDB
- **Cloud IDE**: Goorm IDE
